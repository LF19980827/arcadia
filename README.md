## 1、用户登陆模块

（已测试）

### 1.1 登录
 
​	请求地址 : /login

​	请求参数:

| 参数名   | 类型   | 意义     |
| -------- | ------ | -------- |
| username | String | 用户名   |
| password | String | 登录密码 |

​	响应参数:

| 参数名        | 类型    | 意义                |
| ------------- | ------- | ------------------- |
| errorMessage  | String  | "错误信息"          |
| code          | String  | 状态信息            |
| sessionUserId | Integer | 用户ID（session中） |

eg1:



```json
{
    "code": "success",
}
```

eg2:

```json
{
    "code": "error",
    "errorMessage": "用户名密码格式错误"
}
```
